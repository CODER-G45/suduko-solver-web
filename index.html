How to Play

  1. Fill the grid so that every row, column, and 3x3 box contains the digits 1-9.
  2. Use the number keys 1-9 to fill in cells.
  3. Click on a cell to select it. Use arrow keys for navigation.
  4. Click "Get Hint" for help when stuck.
  5. Click "Solve Puzzle" to automatically solve the puzzle.
